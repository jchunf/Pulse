## A 组 · 核心功能

> 来自用户首轮选择（对应 R-01~R-14）。

| ID | 名称 | 描述 | 依赖数据 | 所属版本 |
|---|---|---|---|---|
| F-01 | 每日/每周趋势图 | 折线 + spark line，显示使用时长、鼠标里程、KPM 等核心指标的近 N 日趋势 | D-A1, D-A2, D-K1, D-M1 | **MVP** |
| F-02 | 应用使用时长排行 | TOP N 应用的时长柱状图/表格，可切换日/周/月 | D-A1 | **MVP** |
| F-03 | 时段热力图 | 24h × 7d 网格，颜色强度 = 活跃度 | D-A1, D-S1 | **MVP** |
| F-04 | 鼠标轨迹可视化 | 密度热力图（v1.1 已交付）；散点层推迟 | D-M2 | v1.1 ✅（密度，A35）|
| F-05 | 实时仪表盘 | 菜单栏 + 主界面显示"当前应用 / 今日时长 / 今日里程 / 按键次数"实时数字 | 实时流 | **MVP** |
| F-06 | 周报/月报导出 | 生成 PDF/PNG/HTML 本地文件 | 聚合数据 | v1.1 |
| F-07 | 指针里程表（戏剧性对比） | 累计毫米→米→公里；对比"北京→广州"、"马拉松 N 圈"、"赤道 x%" | D-M1 | **MVP** |
| F-08 | 键盘热力图 | 画 QWERTY 布局，按频率染色 | D-K2（可选） | v1.2 |
| F-09 | 专注度环形图 | 深度专注 / 多任务 / 碎片化 三段式占比（定义待校准，见 Q-01） | D-A1, D-A2 | v1.2 |
| F-10 | 每日时间带 | 横向条形图显示一天每个时刻在用的应用 | D-A1 | v1.1 |
| F-11 | 连续记录 | GitHub 贡献图风格，每日 ≥ 4h 算一天 | D-A1 | v1.1 |
| F-12 | KPM 峰值 | 每分钟按键次数的峰值与分布 | D-K1 | v1.2 |

## B 组 · 零新权限的有趣扩展

> 基于 A 组已经具备的采集源，不需要额外的系统权限就能做出来。

| ID | 名称 | 描述 | 依赖数据 | 所属版本 |
|---|---|---|---|---|
| F-13 | 窗口切换 Sankey 图 | 应用 A→B 的转移矩阵做成 Sankey，显示常见跳转链路与"工作流形状" | D-A1 | v2.0 |
| F-14 | 应用组合指纹 | 同时打开的窗口集合（"VSCode+Chrome+Terminal+Slack"）识别"工作栈" | D-A3 | v2.0 |
| F-15 | 死亡区域地图 | 屏幕上鼠标从未到达的像素区域（往往呈现出有趣的"未使用地带"） | D-M2 | v2.1 |
| F-16 | 鼠标点击热力图 | 左/右/中键 + 双击分层显示，定位 UI 交互热点 | D-M3 | v2.0 |
| F-17 | 滚轮里程 | 滚轮累计转动距离，里程表子指标 | D-M4 | v1.2 |
| F-18 | 鼠标速度分布（按应用） | 每个应用内的鼠标平均速度，揭示"慢思考 vs 快操作"模式 | D-M2, D-A1 | v2.1 |
| F-19 | 键盘节奏指纹 | 按键间隔分布（不是内容）形成类"打字节奏变异性"曲线；可作为个人节奏指纹 | D-K1 | v2.1 |
| F-20 | 双手均衡度 | 左/右手键区的按键比例，附带人体工学建议 | D-K2（可选） | v2.1 |
| F-21 | 最混乱多任务时刻 | 记录"同一分钟切换最多应用"的瞬间 | D-A1 | v2.0 |
| F-22 | 被动消费时长 | 无输入事件但应用前台且屏幕亮着 → 判为"看视频/读长文/发呆" | D-A1, D-I1 | v1.2 |
| F-23 | 数字鱼骨头 | 一天每分钟活跃度像心电图，午休/晚饭凹陷一目了然 | D-A1, D-I1 | v2.0 |
| F-24 | 数字分身年度报告 | Spotify Wrapped 风格，年底动画化总结（本地生成可分享图片卡） | 全部聚合 | v3.0 |
| F-25 | 里程碑彩蛋 | 累计鼠标里程跨越某数字时弹出动画（跨越太平洋、跑完马拉松、绕赤道 10%） | D-M1 | v1.2 |

## C 组 · 需要新数据源的扩展

> 每条功能附带需要新增的采集权限或 API。某些（如 F-31 Space 切换）依赖私有 API，需评估维护风险。

| ID | 名称 | 描述 | 新增采集 | 所需权限/API | 所属版本 |
|---|---|---|---|---|---|
| F-26 | 空闲/休息统计 | 无事件 > 5 分钟算休息；每日休息次数与时长 | D-I1 | 事件静默检测（无需新权限） | v1.1 |
| F-27 | 开合盖次数 / 开机时长 | MacBook 合盖/开盖累计；单次开机最长持续 | D-S2 | `NSWorkspace` 通知 | v1.1 |
| F-28 | 电池 vs 接电时间占比 | "移动办公 vs 坐班"画像 | D-S3 | `IOPowerSources` | v2.0 |
| F-29 | 硬件负载时序 | CPU/GPU/内存峰值时刻 | D-H1 | `host_statistics64`，低频采样 | v2.1 |
| F-30 | 麦克风/摄像头占用时长 | "今天会议了多久" | D-S4 | 监听 TCC / `AVCaptureDevice` 活跃通知 | v2.0 |
| F-31 | 输入法切换频率 | 中英文切换，多语言画像 | D-S5 | `TISCopyCurrentKeyboardInputSource` | v2.1 |
| F-32 | 剪贴板使用频率 | 只计数 + 大小，**绝不存内容** | D-I2 | `NSPasteboard.changeCount` 轮询 | v2.0 |
| F-33 | 快捷键使用榜 | `cmd+c/v/z/f/s/shift-3/4` 等高频组合 TOP N | D-K3 | CGEventTap（已有） | v1.2 |
| F-34 | 环境光/亮度分布 | "你主要在什么亮度下工作" | D-E1 | `IOServiceGetMatchingService` 读 `AppleLMUController` | v2.1 |
| F-35 | 网络/位置画像 | 当前 WiFi SSID，推断"家/办公室/咖啡店"分布（不传任何地理信息） | D-S6 | `CoreWLAN`（需位置权限，**默认关闭**） | v2.1 |
| F-36 | 日历联动 | 开会时段自动标签，区分"纯深度工作 vs 会议碎片" | D-X1 | EventKit 授权（**默认关闭**） | v2.1 |
| F-37 | Focus Mode 联动 | 系统专注模式切换自动打标签 | D-S7 | `NSProcessInfo` 活动状态 / `DNSD` 通知 | v2.0 |
| F-38 | 通知中断统计 | 今日被通知打断次数 | D-S8 | `UNUserNotificationCenter` / notification db（风险） | v3.0 |
| F-39 | Space/桌面切换 | 用了几个桌面、在哪个桌面停留最久 | D-S9 | `CGSCopyManagedDisplaySpaces`（**私有 API，风险高**） | v3.0（评估中） |

## D 组 · 衍生聚类与动作层

> 不需要新采集，完全基于已有数据做衍生分析 / 交互动作。

| ID | 名称 | 描述 | 所属版本 |
|---|---|---|---|
| F-40 | 晨型/夜型聚类 | 按激活小时分布判定用户作息类型 | v2.0 |
| F-41 | 冲刺/稳定型模式 | 按单次会话时长方差判定工作节奏类型 | v2.1 |
| F-42 | "数字体重"曲线 | 每日事件总数 × 活跃时长的综合健康指标曲线 | v2.0 |
| F-43 | 同比/环比报告 | 本周 vs 上周、本月 vs 上月的差分报告 | v1.2 |
| F-44 | Shortcuts 集成 | macOS 快捷指令可查询"今天用了多久 Xcode" | v2.0 |
| F-45 | 阈值警报 | "屏幕时间超 8h"、"连续 2h 无中断"疲劳提醒 | v1.2 |
| F-46 | 敏感时段快捷键 | 一键标记"接下来 5/15/30 分钟为私密"，不写入任何数据 | **MVP** |
| F-47 | 时段数据清除 | 允许用户选择时间范围删除数据 | v1.1 |
| F-48 | 加密导出/导入 | 整库加密备份到本地文件（跨设备迁移用） | v1.2 |
| F-49 | 自检状态页 | 采集事件数、最近写入时间、DB 大小、各权限状态 | **MVP** |

## 版本路线摘要

| 版本 | 功能集合 | 验证目标 |
|---|---|---|
| **MVP (0.1)** | F-01, F-02, F-03, F-05, F-07, F-46, F-49 | 采集→存储→三件套可视化链路是否跑通 |
| **v1.1** | F-04, F-06, F-10, F-11, F-26, F-27, F-47 | 轨迹可视化 + 报告导出 + 连续激励 |
| **v1.2** | F-08, F-09, F-12, F-17, F-22, F-25, F-33, F-43, F-45, F-48 | 键盘/鼠标深度指标 + 警报/彩蛋 + 备份 |
| **v2.0** | F-13, F-14, F-16, F-21, F-23, F-28, F-30, F-32, F-37, F-40, F-42, F-44 | 扩展采集源 + 动作层 + 衍生画像 |
| **v2.1** | F-15, F-18, F-19, F-20, F-29, F-31, F-34, F-35, F-36, F-41 | 长尾有趣功能 |
| **v3.0** | F-24, F-38, F-39 | 年终 Wrapped 与高风险采集评估 |
